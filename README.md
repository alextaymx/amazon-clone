# Installation Steps

## Using npm

Run commands

1. `npm install`

2. `npm run dev`

## Or using yarn

Run commands

1. `npm install --global yarn`

2. `yarn install`

3. `yarn run dev`

## Git Commands

Run commands

1. `git remote add origin https://github.com/alextaymx/amazon-clone.git`

2. `git remote set-url origin https://github.com/alextaymx/amazon-clone.git`

3. `git add *`

4. `git commit -m 'app build'`
